Components in React: 
--------------------
    ->Components are for the smallest Units in UI(user interface) 

    Types : 
        1. function based components 
            ->Any components declared using Functions is called CBC 
            ->CBC also called as a stateLess components 
            ->here States are not present 
            ->Render() , super , this keywords not present

            Syntax 1: 
                    function ComponentName(){
                            return (
                                here write HTML code
                            )
                    } 
                    export default ComponentName;


            Syntax 2: 
                    const ComponentName=()=>{
                            return (
                                here write HTML code
                            )
                    } 
                    export default ComponentName;




        2. Class Based Components
